What is Programming Language ?

A programming language is a vocabulary and set of grammatical rules for instructing a computer or computing device to perform specific tasks. The term programming language usually refers to high-level languages, such as BASIC, C, C++, Python, Java, ASP.Net, C sharp, and Pascal.
